 ※ 실습 계정 생성 및 실습 데이터 추가
    1. c:\data 폴더 작성
       sql.sql, HR.sql, SCOTT.sql 파일을 c:\data 폴더에 복사


    2. 계정 생성 및 테이블 작성, 데이터 추가
        cmd> sqlplus sys/"java$!" as sysdba
            또는
        cmd> sqlplus / as sysdba

        SQL> @C:\data\sky.sql

        SQL> CONN sys/"java$!"  AS  SYSDBA
        SQL> @C:\data\HR.sql

        SQL> CONN sys/"java$!"  AS  SYSDBA
        SQL> @C:\data\SCOTT.sql

    [참고]
        수업용으로 제공 한
        sky.sql, HR.sql, SCOTT.sql 파일에는
        계정 생성, 권한 설정과 테이블 생성 및 데이터를 추가되도록 작성 되어 있으므로
        따로 계정 생성하지 않고 sys 계정에서 위의 쿼리만 실행하면 된다.


    3. 확인
        SQL> CONN sky/"java$!"
        SQL> SELECT * FROM tab;

        SQL> CONN hr/HR
        SQL> SELECT * FROM tab;

        SQL> CONN scott/TIGER
        SQL> SELECT * FROM tab;


    4. SQL Developer
        1) 오라클 서버에 접속하기 위해 필요한 사항
             (1) 호스트이름 : 오라클 서버가 설치된 서버 IP 주소
                   localhost 또는 127.0.0.1 : 자기자신컴퓨터
             (2) 포트번호 : 기본값으로 설치한 경우 1521
             (3) SID : XE
                   Express Edition 기본 SID : XE
                   Enterprise Edition 기본 SID : ORCL
             (4) 사용자 이름 및 비밀번호

        2) sky 계정을 등록
            (1) 좌측 "접속" 화면에서 + 클릭후 아래를 입력
                 Name : sky계정

                 사용자이름 : sky
                 비밀번호 : java$!

                 호스트이름 : localhost
                 포트 : 1521
                 SID : XE
            (2) 접속

        3) 테스트
            SELECT * FROM tab;


    5. 방화벽 포트 추가
       - 자바 및 외부 컴퓨터에서 접속을 위한 오라클 포트 연결 허용 추가
       - 제어판 > 시스템 및 보안 > Windows Defender 방화벽 > 고급 설정
         1) 인바운드 규칙 - 새 규칙
         2) 포트 - TCP 특정 로컬포트 : 1521 -  연결 허용 - 도메인/개인/컴퓨터 체크 - 이름 : ORACLE
